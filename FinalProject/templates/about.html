    About

    This project was made with the pourpose to build a Website to show my skills as a Full-Stack developer.

    I'm Not A Web-Designer

    For this project I made a Glassmorphism Website based on the Online Tutorials Glassmorphism Landpage video. I did not copy his code, I also created the layout for the other pages in Figma and I added a lot of funtionality to the website.

    Scroll

    For the scrolling effects in this container I used GSAP Library with ScrollTrigger Plugin.

    What's Behind It

    The Back-End was made with Flask(Python) and SQLite3 as the database(to capture leads for marketing porpouse). The Front-End is written in HTML, CSS and JavaScript.

    Why Not Bootstrap?!

    I didn't want to use a CSS framework(Like Bootstrap or Tailwind) so I could challenge myself and do everything from scratch, also to not rewrite the whole UI kits available in the Frameworks.
